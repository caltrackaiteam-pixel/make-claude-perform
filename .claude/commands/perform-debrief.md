Debrief this work session and capture learnings to memory.

## Session Summary

1. **What was accomplished** (bullet list of completed work)
2. **What was not completed** (and why)
3. **Decisions made** (that should be remembered)
4. **Patterns that worked well** (to repeat)
5. **Patterns that failed** (to avoid)
6. **Open questions** (unresolved)

## Memory Capture

For each item worth remembering, save to `~/.claude/projects/[project]/memory/`:

- **Decisions**: Save as `project_[topic].md` with type: project
- **User preferences discovered**: Save as `feedback_[topic].md` with type: feedback  
- **New references found**: Save as `reference_[topic].md` with type: reference

## Output

```
## Session Debrief — [date]

### Shipped
- [item]

### Pending
- [item]: [blocking reason]

### Saved to Memory
- [memory file]: [what was saved]

### Next Session Should Start With
- [recommended first action]
```
